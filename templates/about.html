📖 About SignDetect

Real-Time Sign Language Detection

SignDetect is an advanced web application that leverages cutting-edge computer vision and machine learning technologies to detect American Sign Language (ASL) gestures in real-time. Using your webcam, the system can recognize hand signs and convert them into readable text, making communication more accessible.

Key Features

🎥 Live Detection

Real-time webcam feed with instant sign recognition and confidence scores.

📝 Sentence Builder

Build complete sentences by stringing together detected signs with spacing controls.

🔊 Text-to-Speech

Convert your signed sentences into spoken words with built-in speech synthesis.

📊 Detection History

Track all your detections with comprehensive history and statistics.

🌙 Dark Mode

Eye-friendly dark theme that persists across sessions.

👤 User Accounts

Create an account to save your detection history and track progress.

Technology Stack

SignDetect is built with modern, professional technologies:

🐍 Flask 🗄️ SQLAlchemy 🔐 Flask-Login 👁️ OpenCV ✋ Mediapipe 🧠 TensorFlow/Keras 🎨 Bootstrap 5 ⚡ JavaScript ES6

How It Works

  1. Hand Detection: Mediapipe Hands detects and tracks your hand landmarks in real-time.
  2. Feature Extraction: The system extracts 21 key hand landmark coordinates.
  3. Classification: A trained Keras neural network model classifies the hand gesture into ASL letters.
  4. Live Streaming: Video frames are streamed via MJPEG for smooth, real-time viewing.
  5. Sentence Building: Detected signs are automatically concatenated to form words and sentences.

Whether you're learning ASL, developing accessibility tools, or just interested in computer vision, SignDetect provides a powerful, user-friendly platform for sign language detection.
